## ⚙️ Setup & Run

### 1. Clone project

```bash
git clone https://github.com/your-repo/volunteerhub.git
cd volunteerhub
```

### 2. Run with Docker

```bash
docker-compose up -d
```

### 3. Run Backend manually

```bash
cd backend
./mvnw spring-boot:run
```
